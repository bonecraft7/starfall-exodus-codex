# Repository Structure (Starfall Exodus)

```
starfall-exodus/
├── campaigns/
│   └── save_states/
├── game_files/
│   ├── equipment/
│   │   ├── alien/
│   │   └── terran/
│   ├── locations/
│   └── system/
│       └── engines/
├── docs/
│   └── ai_support/
└── tools/
```

## Conventions
- `.bak` backups collected into `/bak/`.
- `/tools/` holds PowerShell utilities (init, replace, changelog, bootstrap).
- `Jeff_Lore.json` and `Penny_Dreadfulshade.json` anchor personalities.
